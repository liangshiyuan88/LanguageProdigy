# 📚 语文小达人

> 一款专为小学四年级学生设计的语文学习游戏，通过游戏化的方式提高孩子的**作文写作能力**和**阅读理解能力**。

## ✨ 功能简介

游戏包含四大模块，覆盖作文、阅读、词语三大语文核心能力：

| 模块 | 说明 | 题目数量 |
|------|------|----------|
| ✏️ 作文训练 | 7步引导式写作，从审题到完成全程辅导 | 100 篇 |
| 📖 阅读理解 | 短文阅读 + 选择题/填空题，带详细解析 | 100 篇 |
| 🎯 词语游戏 | 成语接龙 + 词语填空，每次随机出题 | 100 + 100 题 |
| 🏆 积分奖励 | 完成练习赚积分，兑换奖励，解锁徽章 | 10 种奖励 / 8 枚徽章 |

## 🚀 快速开始

### 环境要求

- 任意现代浏览器（Chrome、Edge、Firefox、Safari）
- 无需安装任何软件、无需网络连接

## 📖 模块详细说明

### ✏️ 作文训练（100篇，分三级）

作文训练采用**7步引导式写作系统**，不是让孩子一上来就面对空白页面，而是逐步引导：

| 步骤 | 名称 | 内容 |
|------|------|------|
| 1 | 📖 读题审题 | 做选择题确认理解题目要求 |
| 2 | 📝 学范文 | 带段落标注的范文展示 + 学习小贴士 |
| 3 | 🗂️ 列提纲 | 填写开头/中间/结尾的写作要点 |
| 4 | 💎 词语锦囊 | 分类展示好词，点击收藏 |
| 5 | ✏️ 分段写作 | 一段一段地写，每段有引导和提示 |
| 6 | 🔍 检查修改 | 拼接全文 + 自动检查清单 |
| 7 | 🎉 完成评分 | 星级评分 + 详细反馈 + 与范文对比 |

**分级说明：**

| 级别 | 题数 | 特点 |
|------|------|------|
| 🌱 初级 | 35题 | 日常话题（我的一天、我的妈妈、下雨了…），范文 150-200 字，4 步分段写作 |
| ⭐ 中级 | 35题 | 进阶话题（一件难忘的事、长城、刻舟求剑…），范文 200-300 字，4 步分段写作 |
| 🏆 高级 | 30题 | 深度话题（诚信最珍贵、少年中国说、那一刻我长大了…），范文 300-400 字，5 步分段写作 |

### 📖 阅读理解（100篇，分三级）

每篇文章配 4-5 道题（选择题 + 填空题），答题后即时显示对错和详细解析，答对获得 5 积分。

| 级别 | 题数 | 内容特点 |
|------|------|----------|
| 🌱 初级 | 35篇 | 动物科普、自然现象、简单故事（小蚂蚁、含羞草、乌鸦喝水…） |
| ⭐ 中级 | 35篇 | 成语故事、历史人物、科学知识（滴水穿石、盲人摸象、长城…） |
| 🏆 高级 | 30篇 | 深度文言典故、历史事件（少年中国说、草船借箭、负荆请罪…） |

### 🎯 词语游戏（每次随机10题）

#### 🔗 成语接龙

- 题库：100 组成语接龙
- 玩法：看上方的成语，写出以最后一个字开头的下一个成语
- 评分：完全正确 +3 积分，格式正确（首字对、四字成语）+1 积分
- 每次开始随机抽取 10 组，可反复练习

#### 📝 词语填空

- 题库：100 道词语填空
- 玩法：选出正确的字填入句子中的空格
- 评分：答对 +3 积分
- 每次开始随机抽取 10 题，可反复练习

## 🛠️ 技术说明

- **纯前端实现**：HTML + CSS + JavaScript，无需后端服务器
- **数据持久化**：使用浏览器 localStorage 自动保存进度
- **响应式设计**：支持电脑、平板、手机访问
- **零依赖**：不依赖任何第三方库或框架

---

## 📝 使用建议

1. **循序渐进**：建议从初级开始，逐步挑战中级和高级
2. **定时练习**：每天完成 1-2 个练习，保持学习节奏
3. **家长参与**：家长可定期查看孩子的积分和徽章，给予鼓励
4. **奖励兑现**：当孩子兑换虚拟奖励时，家长及时兑现真实奖励
5. **重复练习**：作文可以反复写，阅读理解可以重新做，词语游戏每次随机出题
